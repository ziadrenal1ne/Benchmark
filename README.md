# FOCP Beneficiary Platform Benchmark

A benchmark study of Moroccan beneficiary management platforms with a focus on cybersecurity, secure software development, and security-by-design principles.

## Overview

This repository contains a benchmark report conducted as part of my Cybersecurity Engineering internship at Fondation OCP.

The objective is to analyze existing beneficiary management platforms in Morocco, identify security best practices, and provide recommendations for the design of a secure beneficiary management platform.

> **Disclaimer**
>
> This repository contains only publicly available information, personal research, and original analysis. It does not include confidential, proprietary, or internal information belonging to Fondation OCP.

## Topics Covered

- National benchmark of beneficiary management platforms
- Functional comparison
- Security analysis
- Authentication & Authorization
- RBAC
- OWASP Top 10
- Secure SDLC
- Risk assessment
- Security recommendations
